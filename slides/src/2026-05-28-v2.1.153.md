### 先に押さえるポイント
- `skipLfs` オプションにより Git LFS ダウンロードをスキップできるようになりました。
- Claude Code では、一度の通知で npm グローバルインストールの自動更新ができない場合に通知が表示され、`/doctor` で修正方法が表示されます。
- ステータスラインコマンドでは、`COLUMNS` と `LINES` 環境変数を受け取るようになり、スクリプトがターミナルの幅に合わせた出力を行えるようになりました。

### 変更内容

#### 新機能
- **`skipLfs` オプションの追加**
  - 関連: `github`/`git` プラグイン
  - Git LFS ダウンロードをスキップできるようになりました。
- **一時的な通知の表示**
  - 関連: `/doctor`
  - npm グローバルインストールの自動更新ができない場合に通知が表示されます。

#### 改善
- **ステータスラインコマンドの環境変数受け取り**
  - 関連: `COLUMNS` と `LINES`
  - スクリプトがターミナルの幅に合わせた出力を行えるようになりました。
- **`claude agents` のオートコンプリート**
  - 関連: `claude agents`
  - ネイティブスラッシュコマンドやバンドルスキルを含むオートコンプリートが行われます。

#### バグ修正
- **ステートフルMCPサーバーの再接続ループの修正**
  - 関連: `tools/list`
  - 再接続ループが修正されました。
- **カスタムAPIゲートウェイのトークン問題の修正**
  - 関連: `Anthropic OAuth`
  - カスタムAPIゲートウェイでユーザーのAnthropic OAuthクレデンシャルが渡る問題を修正しました。
