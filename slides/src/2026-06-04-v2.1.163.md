### 先に押さえるポイント
- `requiredMinimumVersion` と `requiredMaximumVersion` のマネージド設定により、Claude Code のバージョンが許可された範囲内であることを保証できます。

### 変更内容

#### 新機能
- **`requiredMinimumVersion` と `requiredMaximumVersion` のマネージド設定の追加**
  - 関連: `settings.json`
  - Claude Code のバージョンが許可された範囲内であることを保証します。
- **`/plugin list` コマンドの追加**
  - 関連: `/plugin list`
  - インストールされたプラグインを簡単に確認できます。
- **`/btw` の "c to copy" ショートカットの追加**
  - 関連: `/btw`
  - 生の Markdown アンサーをクリップボードにコピーできます。

#### 改善
- **Hooks: Stop と SubagentStop フックの拡張**
  - 関連: `hookSpecificOutput.additionalContext`
  - フックから Claude へのフィードバックを提供できます。
- **Skills: `$` エスケープシンタックスの追加**
  - 関連: `\$`
  - コマンドボディ内でリテラルの `$` を含めることができます。

#### バグ修正
- **`claude -p` のハングアップ問題の修正**
  - 関連: `claude -p`
  - バックグラウンドコマンドが終了しない場合にハングアップする問題を修正しました。
- **Bash コマンドの失敗問題の修正**
  - 関連: `ANTHROPIC_API_KEY`
  - `CI=true` の場合に Anthropic API キーが設定されていないと失敗する問題を修正しました。
