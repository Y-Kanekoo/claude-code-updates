### 変更内容

#### 新機能
- **`/model` ピッカーのアップデート**
  - 関連: `ANTHROPIC_BASE_URL`
  - Anthropic ゲートウェイのモデルをリストするようになりました。
- **`claude project purge` コマンドの追加**
  - 関連: `claude project purge`
  - プロジェクトのすべての Claude Code 状態を削除できるようになりました。

#### 改善
- **OAuth ログインの改善**
  - 関連: `claude auth login`
  - ブラウザーのコールバックが localhost に到達できない場合にも機能するようになりました。
- **Auto モードのスピナーの改善**
  - 関連: `--dangerously-skip-permissions`
  - パーミッションのチェックが遅れたときにスピナーが赤くなるようになりました。

#### バグ修正
- **セキュリティの修正**
  - 関連: `allowManagedDomainsOnly` / `allowManagedReadPathsOnly`
  - セキュリティの脆弱性を修正しました。
- **画像の処理の改善**
  - 関連: 画像のアップロード
  - 画像のサイズが大きい場合にセッションが破壊される問題を修正しました。
