### 変更内容

#### 新機能
- **サブエージェントの生成**
  - 関連: `sub-agents`
  - サブエージェントが自身のサブエージェントを生成できるようになりました。
- **Amazon Bedrock の AWS リージョン読み取り**
  - 関連: `~/.aws` コンフィグファイル
  - `AWS_REGION` が設定されていない場合に `~/.aws` コンフィグファイルから AWS リージョンを読み取るようになりました。

#### 改善
- **パフォーマンスの改善**
  - 関連: `long conversations`
  - 長い会話でのパフォーマンスが改善され、冗長なメッセージの正規化が削減されました。
- **アイドル CPU 使用率の削減**
  - 関連: `/goal` ステータスチップ
  - `/goal` ステータスチップの再描画が削減され、アイドル CPU 使用率が低減されました。

#### バグ修正
- **セッションのフリーズ**
  - 関連: `1M context`
  - セッションが 1M コンテキストを使用してフリーズする問題が修正されました。
- **エージェントビューの不具合**
  - 関連: `Working` スピナー
  - エージェントビューがワーカーの応答後も「Working」スピナーを表示し続ける問題が修正されました。
