### 変更内容

#### 新機能
- **`X-Claude-Code-Session-Id` ヘッダー追加**
  - 関連: API リクエスト
  - プロキシがセッションごとにリクエストを集約できるようになりました。

#### 改善
- **VCS ディレクトリ除外リストの更新**
  - 関連: `.jj` と `.sl`
  - Grep とファイルの自動補完が Jujutsu と Sapling のメタデータに遡らないようになりました。
- **メモリ使用量の削減**
  - 関連: ロングセッション
  - マークダウンとハイライトのレンダリングキャッシュが削減され、メモリ使用量が削減されました。

#### バグ修正
- **`--resume` の修正**
  - 関連: セッションの再開
  - v2.1.85 以前のセッションで "tool_use ids were found without tool_result blocks" エラーが発生しないようになりました。
- **ファイル外の書き込みの修正**
  - 関連: プロジェクトルート外のファイル
  - 条件付きスキルやルールが設定されている場合、ファイル外の書き込みが失敗しないようになりました。
