### 変更内容

#### 新機能
- **バックグラウンドセッションの動作変更**
  - 関連: `Ctrl+T` 
  - アイドル時にセッションが終了しないようになり、Claude Code の更新が適用されます。
- **`/code-review` コマンド**
  - 関連: `/code-review`
  - 正確性のバグを特定の努力レベルで報告します。

#### 改善
- **自動アップデーターの改善**
  - 関連: 自動アップデート
  - 一時的なネットワークエラーのリトライ、エラーの詳細な報告、更新失敗時のバージョン表示が改善されました。
- **diff レンダリングのパフォーマンス改善**
  - 関連: 大規模ファイル編集
  - 大規模ファイル編集時の diff レンダリングが高速化されました。

#### バグ修正
- **エンタープライズログイン制限の修正**
  - 関連: `forceLoginOrgUUID`、`forceLoginMethod`
  - 第三者プロバイダーおよび API キーセッションに対するログイン制限が強化されました。
- **`&` 文字のエスケープ修正**
  - 関連: `!` コマンド
  - `&` 文字が `&amp;` として表示される問題が修正されました。
