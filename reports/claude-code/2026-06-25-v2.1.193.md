### 変更内容

#### 新機能
- **`autoMode.classifyAllShell` 設定の追加**
  - 関連: `autoMode.classifyAllShell`
  - Bash/PowerShell コマンドのオートモード分類を強化します。
- **オートモードの却下理由の表示**
  - 関連: `/permissions`
  -却下理由がトランスクリプト、却下トースト、`/permissions` の最近の却下に表示されるようになりました。
- **`claude_code.assistant_response` OpenTelemetry ログイベントの追加**
  - 関連: `OTEL_LOG_ASSISTANT_RESPONSES`
  - アシスタントのレスポンステキストを含むログイベントが追加されました。

#### 改善
- **背景エージェントの改善**
  - 関連: 背景エージェント
  - 背景エージェントの動作が改善され、他のタスクの実行中にエージェントが継続して動作するようになりました。
- **MCP `headersHelper` 認証の改善**
  - 関連: `headersHelper`
  - ツール呼び出しの際の 401/403 エラーに対して、ヘルパーが自動的に再実行および再接続するようになりました。

#### バグ修正
- **`/model` とクライアントデータ依存の UI の修正**
  - 関連: `/model`
  - `/login` の直後に `/model` と他のクライアントデータ依存の UI が古いまたは空の状態で表示される問題が修正されました。
- **背景化の修正**
  - 関連: 背景化
  - 背景化の際に "N 個の背景タスクが破棄されます" というメッセージが表示される問題が修正されました。
